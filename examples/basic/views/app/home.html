Nexio is basic simple MVC framework using golang

Package Nexigo: 

import "github.com/nexigolabs/nexigo"

Features

  • Routing
  • Serve Http
  • Controller

Routing

Example :

import (
	"github.com/nexigolabs/nexigo"
)

func main() {
	nexigo.Route("/", &ctl.HomeController{})
	nexigo.Route("/crud/", &ctl.CrudController{})
	nexigo.Route("/markdown", &ctl.MarkdownController{})
	nexigo.Route("/websocket", &ctl.WebsocketController{})
	nexigo.Route("/sql", &ctl.SqlController{})
	...
}

Serve Http

Example :

import (
	"github.com/nexigolabs/nexigo"
)

func main() {
	...
	nexigo.Run()
}
